# Auto tests for Principles of Computer Organizations course

### Usage
Go to directory with your lab and run following commands:
```
cd ../
git clone https://github.com/ucu-cs/c_strings_auto_testing auto_tests
cd auto_tests
cp -r ../c_str_lib ./
./compile.sh
./bin/gtester
```
